Socket
 - 인터넷에 연결된 다른 컴퓨터에 접속하기 위한 접속구 역할
 - tcp.ip protocol을 사용하여 다른 시스템과 통신하려고 할 때
 - tcp/ip Socket

서버에서 사용되는 send()함수는 
byte단위로 데이터를 전달하기 때문에 encoding을 해줘야한다.

클라이언트에서는 서버에서 보낸데이터를 recv()함수로 받고 다시 decode해준다.

클라이언트, 서버 간의 통신에 ChatMsg 객체 활용
전달된 메시지를 화면에 표시

-------------------------------------------------------------------------------------------------------------------

client 다중 접속을 위하여 Tread사용
이용자가 서버에 접속하면,
채팅기능은 별도의 쓰레드에 의해서 실행되도록 하고
메인 쓰레드는 이용자 접속대기 기능만 수행하도록 한다.

-------------------------------------------------------------------------------------------------------------------

귓속말 기능
서버측에서 데이터를 클라이언트에게 보낼때 필요한 것
 - 소켓
1번 이용자 -> 모든 이용자에게 전송
1번 이용자 -> 2번 이용자에게 전송하려면
msg = soc.recv(1024)
soc1.send(msg)
soc2.send(msg)
soc3.send(msg)
soc4.send(msg)
soc5.send(msg)

서버에서 수신되는 메시지는 루프를 사용하여 대기한다.
클라이언트는 임의의 시기에 메시지를 서버로 전달가능해야 한다.
